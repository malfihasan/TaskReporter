# Welcome to TaskReporter

**TaskReporter** is your personal task and document management system, similar to Notion or ClickUp. It's designed to run locally and can be hosted on your own domain.

## Features

- 📄 **Markdown Pages** - Write and organize documents using Markdown
- ✅ **Task Management** - Track your tasks with status, priority, and due dates
- 🔍 **Full-text Search** - Find anything quickly across all your pages
- 📁 **Folder Organization** - Organize pages in a hierarchical folder structure
- 🌙 **Dark Mode Support** - Easy on the eyes

## Getting Started

1. Create your first page using the **+** button in the sidebar
2. Add tasks to track your work
3. Organize pages into folders

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + K` | Open search |
| `Ctrl/Cmd + N` | New page |
| `Ctrl/Cmd + S` | Save page |
